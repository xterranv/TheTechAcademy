An Anchor tag allows you to link to other pages.

  • Files can be local to the directory where your requesting html files exists (Relative path) or another Directory or website path (Absolute path)(
  • Absolute path - contains a full path consisting of 1.Volume 2. Directory name, and 3. Filename. Example: C:\Projects\HTML\Index.html (Explicit href)
  • Relative Path - File resides in the same directory as the requesting file and thus may only need the filename or subdirectory of the requesting file (Implicit href)
